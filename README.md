WALM MUSM - простой, написанный на Python и работающий с mpv и ffmpeg плеер.

Поддерживает "открыть с помощью" и доступные форматы в виде mp3, ogg, wav, m4a и flac

Для работы потребуется установить ffmpeg и mpv
